Exercícios Javascript

Para cada uma das questões, escreva uma função correspondente no arquivo scripts/main.js. As funções estão nomeadas - de qual questão cada função pertence deve ficar evidente pelo nome. Depois que escrever a função e salvar o arquivo, atualize esta página HTML, e cheque se as células das tabelas abaixo ficaram verdes. Se estiverem vermelhas, sinal que ainda não está correto!

A nota são quantas questões foram corretamente respondidas, dividido pelo número total de questões. Acerte todas para gabaritar!
